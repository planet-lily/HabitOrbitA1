# Social Spark Android App

## Overview
This project is a native Android application developed using **Kotlin in Android Studio**.  
The app helps users maintain social connections by suggesting simple social interactions (“sparks”) based on the **time of day** entered by the user.

---

## Objective
- Provide social interaction suggestions based on user input.
- Use **if statements** to determine appropriate suggestions.
- Implement **error handling** for invalid inputs.
- Use **GitHub for version control** and **GitHub Actions** for automated builds.

---

## Features

### 1. User Interface (UI)
- Title: ** Social Spark System**
- Input field for entering time of day:
  - Example inputs: `Morning`, `Afternoon`, `Dinner`, `Night`
- Display area for showing suggestions
- **Reset button** to clear input and output

---

### 2. Social Spark Suggestions
| Time of Day            | Suggestion |
|-----------------------|------------|
| Morning               | Send a “Good morning” message to a family member |
| Mid-morning           | Send a quick “Thank you” to a colleague |
| Afternoon             | Share a meme or interesting link |
| Afternoon Snack Time  | Send a “thinking of you” message |
| Dinner                | Call a friend/relative for a quick chat |
| Night / After Dinner  | Leave a thoughtful comment on a post |

---

### 3. Error Handling
- Displays messages when:
  - Input is empty
  - Input does not match expected values
- Provides clear feedback to guide correct input

---

### 4. Technologies Used
- **Kotlin**
- **Android Studio**
- **GitHub**
- **GitHub Actions (CI/CD)**

---

## How the App Works
1. User enters a time of day
2. App processes input using **if statements**
3. Matching suggestion is displayed
4. If input is invalid → error message shown
5. Reset button clears all fields
